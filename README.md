# Patapam is a Developer's Institute hackathon project. 
### It has two goals : 

1. Fulfill Hackathon requirements
2. Create a fun tool for my kids

## It contains three main areas : 

1. **Library / Cards**: 
Inspired by "How to teach your baby to read" - Glenn Doman
Cards with voice and greetings to learn words

2. **Mini Games**: 
- Tic-Tac-Toe with two levels
- Memory with three levels and two type of cards, Patapam and his friends, or random Pokemon fetched from API

3. **Learn**: 
Two learning levels (Kita aleph and guimel) with mathematic questions on 3 levels (5 questions each), rating stars and level accessibles only when you succeeded the previous ones. Local storage to keep your progress.
